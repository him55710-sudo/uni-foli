# Open Source Snapshots

This directory stores shallow clones of common repositories that are broadly useful for the `polio` backend.

## Purpose

- quick local reference
- license and implementation review
- future bootstrap material

Optional tools that are not clearly needed across the whole stack should stay in the catalog only.
